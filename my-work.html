Meet App: A Progressive Web Application (PWA)

Meet App is a progressive web application developed with a Test-Driven Development (TDD) approach, where tests are written before implementing the actual functionality in the code.

The application enables users to search for a specific city and retrieve a list of upcoming events hosted in that city. Additionally, users have the capability to install the app on desktop and add the app to their home screen on mobile and access it at any time, even when offline, ensuring continuous access to event information.

Moreover, the app incorporates two graphs to enhance visual appeal and facilitate data analysis. One graph, presented as a scatterplot, illustrates the distribution of events across various locations. The other graph, displayed as a pie chart, visualizes the popularity of event genres, enabling users to draw conclusions more effectively from the data. The technologies and tools used in the development of the Meet app include:

  • Create React App: Used as the template for building the application's frontend.
  • AWS Lambda: Hosting service utilized for deploying the app, featuring a serverless backend architecture.
  • Google Calendar API: Integrated into the app to provide functionality related to calendar events and scheduling.

    Cine-Verse App with React

    Cine-Verse is a dynamic web application designed for movie enthusiasts, providing a comprehensive platform to explore information about different movies, directors, and genres.

    As the sole developer of the Cine-verse app, my primary responsibility was to architect, develop, and style the entire application from scratch, utilizing React for its frontend and enhancing its aesthetics with React-Bootstrap. Key Features Implemented:

  • User Authentication and Profile Management: Users can easily register, log in, and log out. They have control over their profiles, allowing them to update details like their username, email, birthday, and password. Additionally, users can delete their accounts with a simple click.
  • Movie Exploration: The app enables users to access detailed information about different movies. They can filter movies by title, search for specific genres, and view details such as synopsis and movie director.
  • Favorite Movies List: Users can create and manage a list of their favorite movies, which is displayed on their profile page. This feature allows users to easily keep track of their favorite selections.

    Chat App: A React Native Application

    Chat App is a React Native application designed to facilitate real-time communication between users.

    Initially, users authenticate anonymously and can then perform various actions, including sending messages, selecting and sharing images from the device's image library, capturing and sharing photos using the device's camera app, and sharing location data displayed on a map view. Technologies and tools used for this project:

  • Developed using React Native and Expo.
  • Chat conversations are stored in the Google Firestore Database.
  • The chat interface and functionality are implemented using the Gifted Chat library.

    Cine Verse API

    This project involves the development of a server-side application that implements CRUD operations for managing movie data and user information.

    • Express.js framework
    • MongoDB hosted on Atlas
